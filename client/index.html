Basejump: Image Search Abstraction Layer

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

    • Example Search usage:

    https://floating-chamber-51490.herokuapp.com/api/imagesearch/blue?offset=4

    Example Search output

    { "url": "https://farm5.staticflickr.com/4260/35144528785_060cc63125.jpg", "snippet": "bluesfestival2017zoetermeer039-sugar queen the straight blues band usa-be", "thumbnail": "https://farm5.staticflickr.com/4260/35144528785_060cc63125_t.jpg", "context": "https://www.flickr.com/photos/37169520@N06/35144528785" }

    Example Recent Search usage:

    https://floating-chamber-51490.herokuapp.com/api/latest/imagesearch
